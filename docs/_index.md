`crumbls/subscriptions` manages SaaS plans, features, and subscriber usage for Laravel 11/12/13. Polymorphic subscribers, per-plan feature values, trial and grace periods, translatable plan/feature names, and feature-based usage tracking with automatic resets. Payment integration is intentionally out of scope -- wire it up via events or model overrides.
